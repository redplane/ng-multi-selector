Options Description
items List of items which should be displayed inside drop-down list.
display-property Which property of item should be used for displaying item information inside drop-down list.
value-property Which property of item should be used for binding into selected item list.
ng-disabled Whether component should be disabled or not.
placeholder-title Title placeholder.
placeholder-search Search box place holder.
maxlength Search box maximum length.
separator Separator which should be used to separate selected items.
auto-close Whether drop-down menu should be hidden when an item is selected.
limit-item-amount Maximum number of items which can be selected.
is-clear-button-available Whether clear button is available or not.
is-search-box-available Whether search box is available or not.
interval Amount of time that search box emits searching event after input has been entered.
custom-item-template Whether custom item template is supported or not.
active-class Class which is used when item is selected.
